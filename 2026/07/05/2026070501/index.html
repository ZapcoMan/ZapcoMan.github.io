我潜入了一个"消费返利"项目的启动大会——这是我的现场记录

2026-07-05 #网络安全#黑客组织#社工

2025年9月9日，一个叫”盟盟惠”的平台举办了启动大会。我以参会者身份进入现场，全程记录。一年后，我将这份材料整理成文。

一、这个项目是什么？

一句话： 一个以”绿色积分””通证经济””数商模型”为包装的、涉嫌传销+非法集资的资金盘项目。

它的宣传话术非常”高级”——满口都是政策热词：共同富裕、乡村振兴、数字经济、数据要素……但剥开外壳后，内核就是：

听起来很眼熟？是的，这就是庞氏骗局的数学结构，换了一层皮而已。

二、他们是怎么洗脑的？—— 10种话术逐个拆解

我在现场听了整整一天（加晚上一场宴会），整理出以下完整的话术清单：

话术1：权威背书

演讲者自称：

“我们是商务部研究院绿色消费积分制度特聘专家”
“国家市场监管总局数商制度与监管标准特聘专家”

拆解：

  • “特聘专家” ≠ 公务员/官员，通常是课题合作方临时聘用
  • “国家级课题” ≠ 国家政策法规——研究结论不具备法律效力
  • 用个人头衔压人，但拿不出任何红头文件

话术2：政策挪用

引用了真实的重磅文件——《关于构建数据基础制度更好发挥数据要素作用的意见》（俗称”数据二十条”，2022年12月发布），然后做了完全歪曲的解读：

文件实际内容 他们怎么说的
数据要素市场化配置 → 支持”通证经济”
探索数据资产化 → 支持积分兑换股票
数据收益分配制度 → 支持三级分佣模式

事实： “数据二十条”全文从未出现”通证””积分””数商””消费补贴”等词汇。

话术3：焦虑制造

“现在报复性死亡的老板非常多，以前只是自己跳楼，现在是要开车撞人再去自杀。”

用极端个案制造恐慌，然后给出唯一出路：”走消费补贴这条路”。隐藏了所有其他正常选项。

话术4：杯子案例——一套精心设计的逻辑陷阱

这是整场演讲最核心的理论部分，用了将近40分钟来讲一个杯子：

然后说：

  1. 互联网把价格打到了接近成本价 → 中间商被消灭了
  2. 但中间商消失 = 财富分配环节消失了 = 大家赚不到钱了
  3. 所以杯子应该卖10块而不是3块 → 因为商品承载着社会财富分配的使命

漏洞在哪？

  • 商品的功能是满足需求，不是分配财富。工资、税收、社保才是分配机制。
  • 人为抬高物价来维持中间环节，这叫效率倒退，不是分配正义。
  • 真实目的：为恢复多层分销找理论依据。

话术5：概念替换

正常说法 他们怎么说
拉人头 客户分享
上线/下线 批发商 / 零售商
传销 分享经济
入门费 数商执照

通过重新定义词汇来逃避定性。

话术6：煤气灯效应（Gaslighting）

这是全场最高能的一段操作：

“本意是个老鼠会呀。传销的本意原来是庞氏骗局呀。是与不是？是。但我们不能把客户介绍客户就定位成传销啊。不然企业的产品怎么卖？”

四步操作：

  1. 抢先承认——自己先说出你心里的疑虑（”是的，像老鼠会”）→ 让你觉得他很诚实
  2. 偷换概念——“但客户介绍客户≠传销”→ 让你怀疑自己的判断
  3. 污名化质疑者——“不怀好意的人才会把分享说成拉人头”→ 直接将清醒者 扣上了一定巨大的 《不怀好意》 的帽子，让你不敢开口，更加不敢提醒身边的人
  4. 封堵退路——“不然产品怎么卖？”→ 暗示只有这条路

先招供再洗白，降低防御后再重构认知。

那么问题来了——“客户介绍客户”到底是不是拉人头？

动作本身完全合法。 你觉得好东西告诉朋友，天经地义。关键在于利益机制怎么设计：

合法的客户推荐 盟盟惠的做法
层级 只有1层（你推荐朋友，你拿奖励；朋友再推荐别人，跟你没关系） 3个层级（零售→批发→代理），上线从所有下线的业绩中抽成
收入来源 佣金来自商家利润（本来要花在广告上的钱） 佣金来自下线交的钱 / 购买的”资格”
入门门槛 不用付费就能注册和推荐 需要申请”数商执照”，可能涉及费用
主要收入靠什么 推荐只是附带的，正常工作/生意才是主业 收入依赖不断发展下线
流通的是什么 真实商品和服务 积分、通证、”分红权”等虚拟权益

一句话区分：

合法推荐：”产品好，你可以告诉朋友。” → 围绕产品做生意
非法传销：”加入我们，拉人一起创业。” → 围绕拉人做生意

  • 前者流通的是商品，后者流通的是人头和资金——对于后者来说产品只是一个道具。

话术7：类比误导

他类比的东西 实际性质 可比吗？
家电下乡补贴 财政直接补贴 ❌
新能源汽车置换补贴 国库拨款 ❌
种植/养殖补贴 农业财政补贴 ❌
盟盟惠积分 企业自设返利 被强行等同

将政府财政补贴与企业自发营销行为混为一谈。

话术8：群体压力

晚宴时台上有人讲话带动情绪，要求所有人起立一起跟随音乐唱歌。而不愿配合的人会被旁边的人施压站起来。通过肢体动作一致性制造”我们是一伙的了”的心理暗示，孤立不配合的人。

话术9：宏大叙事收尾

“不负韶华，只争朝夕，谱写出绿色消费积分这个行业的新篇章，共同绘就出全民富裕的宏伟蓝图。”

从商业细节突然拔高到”历史使命”级别，让人没时间理性思考——你已经不是在做生意，你在参与”共同富裕”的事业。

小学语文课本说的好， “人有两个宝 双手和大脑 双手会做工 大脑会思考” 你要用你自己的大脑思考，而不是听从他人，放弃思考

三、他们的商业模式到底是什么？

三级分销体系

三个角色，三级收入，全部依赖下线的业绩抽成。

对照《禁止传销条例》第七条：组织者或经营者发展人员，通过对被发展人员以其直接或间接发展的人员数量为依据给付报酬 —— 三级全中。

“绿色积分”是什么？

声称与传统积分不同，可以兑换：

  • ✅ 商品或服务（表面正常）
  • ⚠️ 企业分红权（越界了）
  • ⚠️ 上市公司股票（严重违规）

并在所谓的”价值结算中心”进行资金结算——变相建立积分↔资金的兑现通道。

对照法规：

  • 2017年七部门ICO公告：代币发行融资属于非法金融活动
  • 2021年十部门通知：虚拟货币相关业务活动属于非法金融活动

积分换个名字叫”通证”，换汤不换药。

四、我在现场看到了什么

宴会场景

启动大会当晚（地点：山X省X南市小X河河畔X金国际酒店 这个酒店河对过不远就是X南市的动物园），会场变成宴席厅。几个细节值得记录：

  1. 递烟社交：席间互相递烟，一位大学生双手接烟。在场已加入的人将其解读为**”这群人素质高、有教养”**。

    注意：双手接烟确实说明有礼貌，但礼貌≠合法。问题在于——当人已经投入了信任或金钱后，会下意识抓取所有正面细节来证明自己没选错。这是认知失调在自我修复，不是有效推理。
    笔者最讨厌的就是狗屁的酒桌文化，不仅压迫人的独立意志还在强迫他人服从他人意志！

  2. 目标人群下沉：现场有多名大学生。学生社会经验少、易被权威话术影响，且每个学生背后是一个家庭网络。
  3. 情绪强制同步：台上要求全员起立摇摆舞动，不愿配合者被亲友施压。

人员筛选机制

宴会结束前，组织方的”团队队长”将部分人单独谈话：

  • 对未配合者评价”态度不好”
  • 最终结论：”以后不会再让你参与” = 拉黑排除

他们的筛选逻辑：

跨区域运作线索

谈话中透露：”团队某些人活动结束后要去下一个地点。“

  • 确认信息：下一个会议地点也是一家酒店
  • 巡回式运作，打一枪换一个地方——与宣称的”380城500家商超”矛盾。正规商业布局不需要秘密转移
  • （推测：下一场酒店里可能也有一群和我们一样的”参会者”。这一点没有证据，但结合巡回式招商的模式，可能性不小）

五、”客户介绍客户”到底是不是拉人头？

动作本身完全合法。问题在于利益分配机制怎么设计。

合法的做法

情形A：纯口碑推荐 — 用户觉得产品好告诉朋友，什么都没拿到。100%合法。

情形B：单层推荐奖励 — 你分享了链接，朋友下单，你拿一笔固定佣金。朋友再推荐别人，你拿不到一分钱。拼多多砍一刀、淘宝联盟都是这种模式。合法。

情形C：正规代理商体系 — 有真实货物流转，收入来自进销差价，层级控制在2-3级以内。可口可乐经销商、品牌加盟店都是这种。合法。

非法传销的特征

当以上特征被逐一突破时：

合法做法 传销变形
只有1级返佣 3级或更多层级计酬
佣金来自商家利润 佣金来自新人的入门费
不需要付费参与 需要交费才能获得推荐资格
推荐是附带收入 主要收入来源是发展下线
有真实货物流通 以虚拟权益代替实物交易

盟盟惠的模式：三级分销（零售/批发/代理）+ 层级计酬 + 可能的入门费 + 虚拟权益（分红权/股票）——五条红线全中。

六、企业到底该怎么正常卖产品？

一句话区分：

合法销售 = 围绕产品做生意（”产品好，告诉朋友”）
非法传销 = 围绕人头做生意（”拉人来创业”，产品只是道具）

七、如何抵抗洗脑？—— 实用防御手册

如果你已经被拉去参加类似的会议，或者身边有人深陷其中——以下方法来自真实现场的实战经验。

第一阶段：入场前（预防层）

1. 提前设定底线

在去之前就告诉自己：今天我只观察，不决策，不签字，不转账。

给自己准备一套标准话术：

  • “我需要回家和家人商量一下”
  • “我先看看材料，回头再联系你”
  • “我对这个领域不太了解，需要时间研究”

这些话的作用不是说服对方，是给自己争取离开的时间。

2. 不带多余的东西

  • 银行卡只带一张限额小的
  • 手机提前充好电
  • 不要当众掏出录音设备（蓝牙耳机听歌是最隐蔽的方式）

第二阶段：现场中（物理防御层）

洗脑会场有三个武器：信息轰炸 + 情绪操控 + 群体压力。 逐个应对：

他们做的 你怎么做 原理
连续几小时的演讲轰炸 嘴巴占住——不停吃东西/喝水 物理上无法被互动
高频煽动性语言 耳朵堵上——戴耳机听自己的音乐 建立心理安全区，隔绝声波
要求起立/举手/喊口号 行为变形——站起来但做不协调的动作（如立正） 用荒谬消解仪式感
周围人都在鼓掌/响应 保持静默——不做任何跟随动作 孤立感是他们最怕的
单独约谈施压 复读机模式——只重复一句话（”回去商量”） 不给对方任何可利用的新信息

核心原则：不要试图在现场辩论。

原因很简单：

  • 对方有精心设计的话术体系，你是即兴发挥
  • 现场有群体压力站在他们那边
  • 辩论输了你会自我怀疑（这正是煤气灯效应的触发条件）
  • 辩论赢了也没用——他们会换一个人来继续

沉默和拖延，比辩论有效一万倍。

第三阶段：离场后（认知恢复层）

1. 先做这件事：把所有群消息设为免打扰，但不退群

直接退群 = 暴露立场，可能引发对方的疯狂挽回攻势。
设为免打扰 = 切断信息输入，同时不激化矛盾。

2. 写下来

把你记得的所有内容写出来：

  • 谁讲了什么
  • 承诺了什么收益
  • 让你交了多少钱
  • 有哪些你觉得不对劲的地方

写作本身就是一种反洗脑过程。 当你把口语化的煽动语言变成书面文字时，逻辑漏洞会自动暴露。

3. 用一个终极问题检验一切

“如果没有人再加入了，钱从哪里来？”

任何涉及”拉人””推荐””分享收益”的项目，都逃不过这个问题。

  • 如果答案是”平台补贴” → 问：平台的收入从哪来？
  • 如果答案是”公司利润” → 问：利润够分给那么多人吗？
  • 如果答案开始绕圈子 → 你已经有答案了。

八、家人陷进去了怎么办？

这是最难的部分——因为你要对抗的不是那个项目，而是你家人的信任结构。

不要做的事

行为 后果
当面大吵说这是传销 家人产生防御心理，反而更信项目里的人
强行把钱抢过来/注销账号 关系破裂，他们会在背后说你”不懂趋势”
在家族群里公开撕 把家人推向孤立无援的位置，只能依赖”团队”

可以做的事

第一步：不站到对立面

❌ “这就是骗人的传销！你是不是疯了？”
✅ “我看了一些资料，有些地方不太明白，你能帮我解释一下吗？”

用提问代替指责。让家人自己开口解释模式的时候，很多漏洞会自然浮现。

第二步：问对问题

不要问笼统的问题（”这靠谱吗？”），要问具体的：

  • “我的钱具体投到了哪个账户？公司全名叫什么？”
  • “积分兑换股票——哪家公司的股票？代码是多少？我在行情软件上查查”
  • “数商执照是在哪个部门办的？给我看一下执照照片”
  • “他们说自己是商务部的专家——我去商务部官网查一下这个名字”

大多数传销项目经不起具体追问。

第三步：等待时机

传销项目有一个生命周期：

最好的劝退时机不是刚加入时（那时正兴奋），而是提现出问题的时候。

在那之前，你能做的就是：

  • 保持沟通渠道畅通
  • 记录所有资金流向
  • 不让家人抵押房产/借高利贷
  • 保存好所有证据（截图、合同、聊天记录、转账凭证）

九、如何举报？—— 以自身安全为前提

⚠️ 第一原则：保护好自己再行动

风险等级 你的情况 建议
低 只是参加过会议，没投钱没拉人 可以匿名举报
中 家人参与其中，你掌握部分证据 匿名举报，注意保护家人隐私
高 你本人曾投入资金或发展过下线 咨询律师后再决定——举报者本身也可能被调查
极高 组织方已经注意到你的”异常行为” 暂缓举报，优先确保人身安全

2. 保护好自己的信息

  • 举报时使用匿名渠道（见下表）
  • 如果必须实名，可以要求个人信息保密
  • 不要在社交媒体上公开发布举报内容——这可能打草惊蛇
  • 保留原始证据的备份（云端加密存储），不要只留一份

3. 心理准备

  • 举报后可能短期内看不到结果（这类案件调查周期长）
  • 可能会有知情者向组织方泄露你举报的信息
  • 家人如果深度参与，可能会与你产生严重冲突
  • 这些都是正常成本——在按下发送键之前想清楚能不能承受
